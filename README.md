# Romaja Helper

A VS Code extension helper.

## Features

- Basic extension structure ready for customization

## Requirements

- VS Code 1.74.0 or higher

## Installation

1. Clone this repository
2. Run `yarn install` to install dependencies
3. Press `F5` to open a new window with your extension loaded
4. Run the command `Hello World` from the command palette (Cmd+Shift+P / Ctrl+Shift+P)

## Development

- Run `yarn compile` to compile TypeScript
- Run `yarn watch` to watch for changes and compile automatically
- Run `yarn type-check` to check for type errors without emitting files
- Run `yarn lint` to check for linting errors
- Run `yarn lint:fix` to automatically fix linting errors

## TypeScript Configuration

This project uses the strictest TypeScript configuration with all strict type checking options enabled, including:
- Strict null checks
- No implicit any
- No unused locals/parameters
- No implicit returns
- No unchecked indexed access
- Exact optional property types
- And more...

This extension currently has no settings.
